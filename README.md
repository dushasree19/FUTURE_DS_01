## Task Objective
The objective of this task is to analyze e-commerce sales data and create an interactive dashboard to help businesses understand their sales performance and make data-driven decisions.

## Tools Used
- Power BI
- CSV / Excel Dataset
- Data Visualization & Analysis

## Dataset Description
The dataset contains e-commerce transaction details including:
- Invoice Number
- Product Description
- Quantity
- Invoice Date
- Unit Price
- Customer ID
- Country

## Dashboard Features
- Total Sales, Total Orders, and Total Customers KPIs
- Sales Trend Analysis over time
- Top-performing products by sales
- Country-wise sales distribution

## Key Insights
- United Kingdom is the highest contributing country in terms of sales
- Sales show a steady increasing trend over the years
- A small number of products contribute significantly to total revenue
- Customer engagement is strong based on order volume
